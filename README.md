# baseball

## 기능 목록
- [ ] 게임 시작 메시지 출력 `숫자 야구 게임을 시작합니다.`
- [ ] 랜덤 숫자 3자리 생성
- [ ] 사용자 숫자 입력 `숫자를 입력해주세요: {입력값}`
  - [ ] 잘못된 값을 입력한 경우 에러 처리 후 종료
- [ ] 맞춘 숫자에 따른 처리
  - [ ] 같은 수가 같은 자리에 있으면 `n스트라이크`
  - [ ] 같은 수가 다른 자리에 있으면 `n볼`
  - [ ] 같은 수가 전혀 없으면 `미스`
- [ ] 사용자가 3개를 모두 맞추면 성공 및 재시작 안내
  - [ ] `3개의 숫자를 모두 맞히셨습니다!`
  - [ ] `게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.` `\n입력`
- [ ] 랜덤 숫자 생성부터 끝까지 반복
